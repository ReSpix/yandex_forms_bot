Проект запускается с помощью Docker compose из директории с файлом docker-compose.yml. 
В Яндекс Формы вносятся 2 интеграции. 
Скриншоты интеграций представлены на изображениях "Интеграция 1" и "Интеграция 2".
Обязательно в первой интеграции переменная "Ответы на все вопросы" должна иметь "Фильтры: JSON"
Текст интеграций для копирования в файле forms_integration.txt. 
Обязательно нужно написать верный id чата как в интеграции, так и в файле bot/key.py. 
Изменить токен бота можно также в файле bot/key.py.
